# Gestion d'Événements - Aurillac

Une application web permettant de centraliser et de gérer les événements à Aurillac. Ce projet offre des fonctionnalités adaptées aux annonceurs, aux visiteurs et aux administrateurs.

## Fonctionnalités

### Annonceurs
- Création, modification et suppression d'événements.
- Interface simple et intuitive pour modifier les événements.

### Visiteurs
- Consultation des événements classés par catégories et sous-catégories.
- Recherche d'événements selon des critères (catégorie, etc.).

### Administrateurs
- Supervision des utilisateurs et gestion des rôles (annonceur/administrateur).
- Validation des événements ou bloquage d'annonceurs ou suppression.
- Analyse des statistiques des visites et exportation des données (CSV, ZIP).

## Technologies Utilisées
- **Backend :** PHP 8, MySQL
- **Frontend :** HTML5, CSS3, Bootstrap 5, JavaScript
- **Autres :** ZipArchive pour les exports, PDO pour la gestion sécurisée des requêtes.
